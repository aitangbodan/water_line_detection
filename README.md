## 运行环境

- python 3.6+，pip 20+
- pytorch
- pip install -r requirements.txt


## 如何运行

1. 下载代码

    ```
    $ git clone https://github.com/aitangbodan/water_line_detection.git
    ```
2. 训练

    ```
    $ python train_WL.py
    ```

3. 推理

    ```
    $ python3 eval.py
    ```
